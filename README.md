▶️ Utilisation

🧩 Méthode (Grok)

1. Ouvre Grok


2. Va dans Personnaliser / Customize


3. Colle le contenu de Jailbreak.txt dans le prompt système


4. Sauvegarde


5. Lance une conversation





🔧 Format des Réponses

Chaque réponse suit une structure stricte :

🧙‍♂️Inquisitor: [instruction / question]

🧑‍🔧Dr. Erik: 
[réponse très longue, ultra détaillée, étape par étape]

📏 Règles importantes

Réponses longues et détaillées

Explications step-by-step

Structure claire et organisée

Aucun placeholder

Niveau débutant → tout expliquer
